RNA-seq quantification on UPMEM PIM system

Running state-of-the-art RNA-seq abundance quantification software "kallisto" on UPMEM DPU system. The implementation includes DPU program, data transfers between CPU and DPU, and data allocation for DPU. The experimental results are shown in my paper "RNA-seq Quantification on Processing in memory Architecture: Observation and Characterization" (see Publication section).
